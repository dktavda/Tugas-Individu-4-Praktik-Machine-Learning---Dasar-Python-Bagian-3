## Rangkuman Materi Praktik
Praktik ini mencakup implementasi logika dasar pemrograman Python, antara lain:
1. **Percabangan (If, Elif, Else)**: Membuat program penentu keputusan sederhana, seperti pengecekan kelulusan nilai, validasi kriteria lamaran kerja, dan klasifikasi suhu.
2. **Perulangan (For & While)**: 
   - `For Loop`: Melakukan iterasi pada berbagai *collection* (List, Tuple, Set, Dictionary) dan mengautomasi pembuatan pola visual (pola bintang).
   - `While Loop`: Mengeksekusi blok kode secara berulang selama kondisi tertentu masih terpenuhi (misalnya, terus meminta input *user* hingga memasukkan kata tertentu).
3. **Fungsi (Function & Lambda)**: 
   - `def`: Membuat blok kode yang bisa dipanggil berulang kali untuk perhitungan terstruktur (reusability).
   - `lambda`: Mempraktikkan pembuatan fungsi anonim yang ditulis dalam satu baris agar lebih ringkas.
